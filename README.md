# UU SwiftTransit

## Uttara University Bus Tracking & Scheduling System

UU SwiftTransit is a web-based bus tracking and scheduling system designed to help students and staff monitor bus locations, view schedules, and receive transportation updates in real time.

The system improves transportation efficiency, reduces waiting times, and provides administrators with tools to manage routes, schedules, and drivers effectively.



## Features

### User Features
- Home Page
- About Page
- Contact Page
- User Registration (Sign Up)
- User Login
- Real-Time Bus Tracking
- Bus Schedule Viewing
- Route Information
- Feedback
- Live Transportation Updates

### Admin Features
- Manage Bus Routes
- Manage Schedules
- Manage Drivers
- Update Bus Status
- Monitor Transportation Operations


## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
