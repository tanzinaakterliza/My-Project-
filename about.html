UU Bus Tracking & Scheduling System


The UU Bus Tracking & Scheduling System is designed to help students and staff track buses, view schedules, and receive real-time updates.

It improves transportation efficiency, reduces waiting times, and provides admins with tools to manage routes, schedules, and drivers effectively.

Project Goals

Provide real-time bus positions
Manage schedules easily
Assist drivers with updates

Expected Deliverables

Frontend UI
Database Design (ER Diagram)
Mock API (optional)
